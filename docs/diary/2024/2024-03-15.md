## 今日学习 Learnings

### Vue 3 Composition API

今天深入学习了 Vue 3 的 Composition API，特别是 `ref` 和 `reactive` 的区别：

- `ref` 适合基本类型数据
- `reactive` 适合对象类型数据
- 两者都可以使数据具备响应式

```javascript
import { ref, reactive } from 'vue'

const count = ref(0)
const state = reactive({ name: 'diary', year: 2024 })
```
