## 今日学习 Learnings

### VitePress

VitePress 是一个基于 Vite 的静态站点生成器，非常适合用来搭建文档和博客。

**主要特点：**
- 🚀 极速的开发服务器（基于 Vite）
- 📝 以 Markdown 为中心
- 🎨 美观的默认主题
- 🔍 内置全文搜索

### GitHub Actions 自动部署

通过 GitHub Actions，每次 `push` 到 `main` 分支时，站点会自动构建并部署到 GitHub Pages。
这意味着只需要提交 `.md` 文件，就能在几分钟内看到更新！

## 如何添加新日记

1. 在 `docs/diary/YYYY/` 目录下创建 `YYYY-MM-DD.md`
2. 按模板填写内容
3. `git add`, `git commit`, `git push`
4. 等待 GitHub Actions 自动部署完成
5. 通过链接访问：`https://YuanfengYan.github.io/diary/diary/YYYY/YYYY-MM-DD`
