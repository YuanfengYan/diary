# 📔 Diary

Welcome to my daily diary. Each entry is a markdown file organized by year and date.

## How to Create a New Entry

1. Navigate to the `docs/diary/YYYY/` folder (create the year folder if needed)
2. Create a new file named `YYYY-MM-DD.md`
3. Write your diary entry using Markdown
4. Commit and push — the site will auto-deploy

## Entry Format

```markdown
# YYYY-MM-DD

## Today's Highlights
- ...

## Learnings
- ...

## Thoughts
...
```
